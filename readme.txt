== Installation ==

1. Upload `alixcan_more_ads` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure the plugin in the 'Options' (or 'Settings') screen by following the onscreen prompts.

## Documentation
If you need more help installing and configuring the plugin, [see here for more information](http://www.alixcan.net/?p=1505).
